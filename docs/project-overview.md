# Project Overview

This project was developed as part of the **Bradesco & DIO Generative AI Bootcamp** and focuses on the use of **Generative AI as a learning support tool**.

The main objective is to create a **thematic study guide using NotebookLM**, centered on the fundamentals of **personal financial planning**. The project explores how Generative AI can assist in organizing knowledge, summarizing curated content, and supporting critical thinking through prompt variation.

The chosen topic addresses practical and accessible financial concepts that are relevant to a broad audience, especially beginners interested in improving their financial organization and decision-making.
